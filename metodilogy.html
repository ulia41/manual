1. ОБЩАЯ ПОСТАНОВКА ЗАДАЧИ

Имеется бассейн отработавшего ядерного топлива с заданной мощностью остаточных тепловыделений.

Требуется определить основные конструктивные (геометрические) характеристики теплообменника, способного отвести заданную мощность при заданном температурном режиме –поверхность теплообмена, длину и объем трубного пучка.

3. КОНСТРУКТИВНЫЙ РАСЧЕТ

3.1 Необходимый общий расход охлаждаемой воды.

Уравнение теплового баланса:

N = G1(t1 вх - t1вых) * сp1 (3.1.1)

Из уравнения (3.1.1) выражается расход воды:

G1 = N / Δt1 * cp1 (3.1.2)

где N – мощность (Вт) (таблица 2.1);

Δt1 - «горячий» температурный перепад, °С (2.3.1)

cp1 - изобарная теплоемкость «горячего» потока, Дж/(кг*град) (2.4.5)

G1 = 3 * 106 / 10 * 4178 = 71,8 кг/с

3.2 Число теплообменных трубок

Пусть предварительное число теплообменных трубок – nmp. Расход через одну трубку определяется по формуле:

Gmp = G1 /nmp (3.2.1)

где nmp – число теплообменных трубок.

Из (3.2.1) число теплообменных трубок находится по формуле:

nmp = G1 / Gmp (3.2.2)

Средняя скорость воды в трубке находится по формуле:

ωmp = Gmp / ρ * Smp (3.2.3)

где ρ - плотность «горячего» потока, кг/м3 (2.4.3)

Smp - площадь проходного сечения трубки, м2 (2.5.2)

Из формулы (3.2.3) выражается расход через одну трубку Gmp и подставляется в формулу (3.2.2).

Число теплообменных трубок в первом приближении корректируется так, чтобы скорость воды была в установленных пределах.

Если ω º - рекомендованное значение скорости воды в трубке (таблица 2.1), то число трубок определяется по формуле:

nmp1 = G1 / ω º * ρ * Smp(3.2.4)

nmp1 = 71,8 / 1,4 * 992,5 * 1,327 * 10-4 = 389

Для квадратного коридорного трубного пучка число трубок корректируется следующим образом. Определяется предварительное число трубок на стороне квадрата:

a = √nmp(3.2.5)

a = √389 = 19,7

Определенное таким образом число трубок в ряду округляется до целого четного числа в сторону увеличения:

a = 20

Этим округлением создается запас количества трубок на аварийное заглушение ( ~ 10%).

Второе приближение общего количества трубок определяется возведением в квадрат уточненного числа трубок в ряду:

nmp2 = a2(3.2.6)

nmp2 = 202 = 400

3.3 Скорость воды в трубке

Окончательное значение скорости воды в трубке находится по формуле:

ω1 = G1 / nmp2 * ρ1 * Smp (3.3.1)

ω1 = 71,8 / 400 * 992,5 * 1,327 * 10-4 = 1.36 м/с

3.4 Критерии подобия и коэффициент теплоотдачи с внутренней стороны

Согласно исходным данным компановка трубного пучка - квадрат (коридорное расположение) (таблица 2.1). На рисунке 3.5.1 представлена геометрия трубного пучка квадратного (коридорного типа).

Пусть трубный пучок образует прямоугольный пакет (коридорное расположение) с относительным шагом:

χ = S / dmp2(3.5.1)

dmp2 – наружный диаметр трубки, м (таблица 2.1).

χ = 0.02 / 0.016 = 1.25

Сторона квадратной ячейки в каждой трубке определяется по формуле:

h2 = χ * dmp2 (3.5.2)

h2 = 1.25 * 0.016 = 0.02 м

Площадь одной ячейки равна:

h22 = 0.022 = 0.0004 м2 (3.5.3)

Полная площадь сечения трубного пучка определяется по формуле:

Sполн = nmp * h22 (3.5.4)

где nmp - количество ячеек (равно количеству трубок) (3.2.6)

h22 - площадь одной ячейки, м2 (3.5.3).

Площадь проходного сечения межтрубного пространства равна полной площади сечения трубного пучка за вычетом суммарной площади, занятой трубками, а суммарная площадь, занятая трубками, равна произведению числа трубок на площадь сечения трубки по внешнему диаметру:

S2 = nmph22 - nmp * (πd2mp2 / 4) = nmp(h22 - πd2mp2 / 4) = nmp * d2mp2 * (χ2 - π / 4) (3.5.5)

S2 = 400 * 0.0162 * (1.252 - 3.14 / 4) = 0.0796 м2

Считаем, что трубный пучок помещен в квадратную в плане обечайку.

Площадь сечения обечайки в первом приближении равна Sполн = nmp * h22, следовательно сторона (квадрата) равна √nmp*h22 = h2 * √nmp . Соответственно периметр квадрата равен учетверенной длине стороны квадрата:

Π = 4 * h2 * √nmp (3.5.6)

Полный смоченный периметр системы, состоящей из трубного пучка и обечайки, равен сумме периметра обечайки и сумме длин внешних окружностей (периметров) всех трубок:

Π = 4 * h2 * √nmp + nmp * π * dmp2 (3.5.7)

Π = 4 * 0.02 * 20 + 400 * 3.14 * 0.016 = 21.7 м

Гидравлический диаметр межтрубного пространства, согласно определению гидравлического периметра, равен отношению учетверенной площади проходного сечения к смоченному периметру:

dг2 = 4 * S2 / Π (3.5.8)

dг2 = 4 * 0.0796 / 21.7 = 0.0147 м

3.10 Определение требуемой величины поверхности теплообмена

3.10.1 Теоретические основы Исходное уравнение теплопередачи имеет вид:

N = κ * F * Δt , (3.10.1)

где N - передаваемая тепловая мощность, Вт (таблица 2.1);

κ - коэффициент теплопередачи, усредненный по площади поверхности, Вт/(м*град) (3.9.1);

Δt - температурный напор теплопередачи, °С.

Если «горячий» и «холодный» температурные напоры равны, то температурный напор теплопередачи определяется как Δt = θ1 = θ2.

Так как «горячий» и «холодный» температурные напоры (θ1 - 2.3.3 и θ2 - 2.3.4) не равны между собой, то в качестве температурного напора теплопередачи используется так называемый средне логарифмический температурный напор.

Из температурных напоров θ1 и θ2 определяются больший и меньший θб и θм . Так как θ1 =10°С и θ2 =12°С, следовательно θб =12°С и θм =10°С.

Средне логарифмический температурный напор определяется по формуле:

t = θб - θм / ln*(θб/θм)(3.10.2)

t = 12-10 / ln*(12/10) = 10.97°С

Из (3.10.1) находится площадь теплопроводящей поверхности:

F = N / (κ*t) (3.10.3)

F = 3000000 / (1920.86 * 10.97) = 142.37 м2

3.11 Определение конструктивных характеристик теплообменника

По величине поверхности теплообмена F (3.10.3) определяется требуемая длина теплообменных трубок. Поверхность теплообмена можно найти по формуле:

F = nmp(πdmp2 * Lmp ) (3.11.1)

Из формулы (3.11.1) можно выразить длину теплообменных трубок:

Lmp = F / (nmp * π * dmp2) (3.11.2)

Lmp = 142.37 / (400*3.14*0.016) = 7.1 м

Определенная таким образом длина трубки является основой для окончательного определения габаритов и массы теплообменника.

Если полученная длина больше 8-9 м, целесообразно конструкцию темлообменника сделать поворотной (с U-образными трубками) или секционной.

Объем трубного пучка находится по следующей формуле:

V = Lmp * nmp * h22 (3.11.3)

где Lmp - длина теплообменных трубок, м (3.11.2);

nmp - число теплообменных трубок (3.2.6);

h22 - площадь одной ячейки, м2 (3.5.3).

V = 7.1 * 400 * 0.0004 = 1.136 м2

Далее в пределах вычисленной величины поверхности теплообмена можно варьировать соотношение между числом трубок и их длиной.

4.1 Водяные эквиваленты потоков

Водяные эквиваленты обоих потоков рассчитываются через массовые расходы и изобарные теплоемкости по следующим формулам:

W1 = cp1 * G1, (4.1.1)

где cp1 - изобарная теплоемкость охлаждаемой воды, Дж/(кг*град) (2.4.5);

G1 - расход охлаждаемой воды, кг/с (3.1.2).

W1 = 4178 * 71.8 = 299980.4 Вт/град

W2 = cp2 * G2, (4.1.2)

где cp2 - изобарная теплоемкость охлаждающей воды, Дж/(кг*град) (2.4.6);

G2 - расход охлаждающей воды, кг/с (3.6.1).

W2 = 4178.5 * 59.83 = 249999.655 Вт/град

Если водяные эквиваленты оказываются строго равны W1=W2, то распределение температур имеет вид прямых, попарно соединяющих входные и выходные температуры.

Так как водяные эквиваленты не равны между собой (W1 > W2 ), следовательно, проводятся дальнейшие вычисления и построения графика.

4.2 Показатель экспоненты для противотока:

p = (1/W1 - 1/W2) * κ * F, (4.2.1)

κ - коэффициент теплопередачи, Вт/(м2 * град) (3.9.1);

F - площадь поверхности теплообмена, м2 (3.10.3);

W1, W2 - водяные эквиваленты потоков, Вт/град (4.1.1); (4.1.2)

p = (1/299980.4 - 1/249999.655) * 1920.86 * 142.37 = -0.182256

4.3 Выходные температуры охлаждаемого и охлаждающего потоков воды:

t1-вых = t1вх - ((t1вх-t2вх) * (1-exp(-p)) / (1-(W1/W2)*exp(-p)) (4.3.1)

где t1вх - температура отводимой из бассейна (охлаждаемой) воды на входе в теплообменник, °С (таблица 2.1);

t2вх - температура промконтурной (охлаждающей) воды на входе в теплообменник, °С (таблица 2.1);

p - показатель экспоненты для противотока (4.2.1);

W1, W2 – водяные эквиваленты потоков, Вт/град (4.1.1); (4.1.2).

t1-вых = 44 - (44-22)*(1-exp(0.182256)) / (1 - (299980.4/249999.655)*exp(0.182256) = 34.002 °C

t2-вых = t2вх + (W1/W2)(t1вх - t1-вых),(4.3.2)

где t1-вых - выходная температура, рассчитанная по формуле (4.3.1), °С.

t2-вых = 22 + (299980.4/249999.655)*(44-34.002) = 33.997 °C

Сравниваем полученные значения выходных температур t1-вых и t2-вых с заданными в поставке задачи t1вых и t2вых и определяем погрешность:

t1-вых = 34.002°C > t1вых = 34°C - погрешность составляет ±0,002°C

t2-вых = 33.997°C < t2вых = 34°C - погрешность составляет ±0,003°C

4.4 Построение проверочных графиков

Так как водяные эквиваленты W1 и W2 не равны друг другу, следовательно, зависимости температур от координаты описываются следующими формулами:

Для «горячего» теплоносителя:

t1(z) = t1вх - (t1вх-t2вх)*(1 - exp(-p(z/L)) / (1 - (W1/W2)*exp(-p)), (4.4.1)

где L – длина теплообменной трубки (трубного пучка), м (3.11.2);

z – текущая координата 0 ≤ z ≤ L с шагом Δz.

t1(z) = 44 - (44-22)*(1-exp(0.182256*(z/7.1)) / (1 - (299980.4/249999.655)*exp(0.182256)

Для «холодного» теплоносителя:

t2(z) = t2вх + (W1/W2)(t1(z) - t1-вых), (4.4.2)

где t1(z) - текущее значение «горячего» теплоносителя, °С (4.4.1).

t2(z) = 22 + (299980.4/249999.655)*(t1(z) - 34.002)

Для решения уравнений (4.4.1), (4.4.2) с переменной z (0≤z≤L ) задается шаг Δz. В данном случае зададим шаг Δz = L/14 =0.507. Тогда zi = zi-1 + Δz.

Для каждого текущего значения zi вычисляется по формуле (4.4.1) очередное ti1(z), далее это значение подставляется в формулу (4.4.2) и вычисляется значение ti2(z).
